Database Files

Important: The database files are required for full functionality but are not included in the main repository due to their size.

Main Raman Database (Required)

  • File: RamanLab_Database_20250602.pkl
  • Size: ~200MB
  • Contains: 6,939+ experimental Raman spectra
  • Sources: RRUFF, WURM, SLOPP/E databases
Download from Zenodo

Installation Location:

Place the database file in one of these locations:

  • Windows: C:\Users\<YourUsername>\Documents\RamanLab_PySide6\
  • macOS: ~/Documents/RamanLab_PySide6/
  • Linux: ~/Documents/RamanLab_PySide6/
  • Alternative: Same folder as the Python scripts

Mineral Modes Database (Optional)

  • File: mineral_modes.pkl
  • Size: ~50MB
  • Contains: Mineral vibrational mode data

This database is included in the main repository.

Installation Guide

Step 1: Install Python

RamanLab requires Python 3.8 or higher. Download from:

Python.org

Step 2: Create Virtual Environment (Recommended)

python -m venv ramanlab_env

Activate the environment:

macOS/Linux:

source ramanlab_env/bin/activate

Windows:

ramanlab_env\Scripts\activate

Step 3: Install Dependencies

pip install -r requirements_qt6.txt

Step 4: Verify Installation

python check_dependencies.py

Step 5: Download Database

Download the database file from Zenodo and place it in the appropriate location (see above).

Step 6: Launch RamanLab

python main_qt6.py
Success! RamanLab should now be running. See the Documentation for usage instructions.
