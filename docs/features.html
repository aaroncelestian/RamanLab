Features

Comprehensive tools for advanced Raman spectroscopy analysis

Core Analysis Tools

üî¨ Peak Fitting

Advanced multi-peak fitting capabilities with multiple models:

  • Lorentzian - Classic peak shape for Raman spectroscopy
  • Gaussian - Symmetric peak fitting
  • Voigt - Convolution of Lorentzian and Gaussian
  • Asymmetric Voigt - 5-parameter model with separate left/right widths
  • Pseudo-Voigt - Efficient approximation for complex peaks
New in v1.3.0: Complete asymmetric Voigt implementation with dual FWHM visualization and enhanced CSV export with separate asymmetric parameter columns.

üìä Baseline Correction

Multiple baseline subtraction methods for optimal background removal:

  • ALS (Asymmetric Least Squares) - Adaptive baseline fitting
  • Polynomial - Flexible polynomial baseline fitting
  • Rolling Ball - Morphological baseline estimation
  • Linear - Simple linear baseline
  • Spline - Smooth spline-based baseline
  • Manual Adjustment - Interactive baseline correction

All methods include real-time preview with instant parameter adjustment.

Comprehensive database system with 6,939+ reference spectra:

  • RRUFF Database - Mineral Raman spectra

Features include:

  • Fast search by name, chemical family, or metadata
  • Correlation-based spectrum matching
  • Export filtered subsets
  • Database statistics and visualization
  • Import custom spectra

üîç Search/Match

Intelligent spectrum identification and matching:

  • Correlation-based Search - Fast similarity matching using Pearson correlation
  • DTW (Dynamic Time Warping) - Advanced alignment for shifted spectra
  • Combined Methods - Hybrid approach for optimal results
  • 6,939+ Reference Library - Comprehensive database of minerals, organics, and plastics
  • Filtered Search - Search within specific chemical families or categories
  • Top Match Display - Ranked results with correlation scores
  • Overlay Comparison - Visual comparison with reference spectra
Powerful Identification: Quickly identify unknown materials by matching against curated reference databases from RRUFF, WURM, and SLOPP/E.
