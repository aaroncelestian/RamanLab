# RamanLab Website

This folder contains the official website for RamanLab - Advanced Raman Spectroscopy Analysis Suite.

## Files

- `index.html` - Homepage with overview and key features
- `features.html` - Detailed feature descriptions
- `documentation.html` - Installation and usage guide
- `download.html` - Download instructions and version information
- `contact.html` - Contact information and support resources
- `styles.css` - Stylesheet for all pages

## Viewing the Website

### Local Preview

Open `index.html` in your web browser to view the website locally.

### Deployment

This website can be deployed to any static hosting service:

- **GitHub Pages**: Push to a `gh-pages` branch
- **Netlify**: Drag and drop the WebPage folder
- **Vercel**: Connect your repository
- **AWS S3**: Upload as static website
- **Any web server**: Upload files via FTP/SFTP

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with gradient accents
- **Cross-browser Compatible**: Works in all modern browsers
- **Fast Loading**: Minimal dependencies, pure HTML/CSS
- **SEO Friendly**: Proper meta tags and semantic HTML

## Customization

To customize the website:

1. **Colors**: Edit CSS variables in `styles.css` (`:root` section)
2. **Content**: Edit HTML files directly
3. **Images**: Add images to the WebPage folder and reference them in HTML
4. **Favicon**: Add `favicon.ico` to the WebPage folder
